Explicación de etiquetas

                

                La etiqueta <code> se utiliza en HTML para
                indicar fragmentos de código cortos, en línea.
                

            
                

                La etiqueta <nav> se utiliza en HTML para
                indicar dónde está ubicada la barra de navegación de la página.
                

            
                

                La etiqueta <p> se utiliza en HTML para
                escribir un párrafo dentro de la página
                

            
                

                La etiqueta <a> se utiliza en HTML para
                escribir link externo o interno en tu página web
                

            
                

                La etiqueta <img> se utiliza en HTML para
                insertar un imagen en tu página web
